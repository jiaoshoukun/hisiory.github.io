python pip 安装

最近刚开始在电脑上安装Python，装完pip之后想要装pillow，结果pip命令不能使用，使用提示命令还是会循环报错。网上找了很多也不对，最终使用如下方式解决！！

1.在命令行定位到pip所在文件夹下，例如我的pip在 D:\Python\Scripts

2.加上python pip3.exe

3.再加 python -m pip install –upgrade pip 出现如下界面即为成功

4.最后直接 pip install 就OK啦
